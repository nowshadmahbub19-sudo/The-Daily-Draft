# The Daily Draft

Full-stack blog application built with Node.js, Express, EJS, and SQLite.

## Features
- Create blog posts
- Edit and delete posts
- View all posts
- Server-rendered pages with EJS
- SQLite database integration

## Stack
Node.js, Express.js, EJS, SQLite, better-sqlite3

## Purpose
Built to practice backend fundamentals and full-stack CRUD workflows.

## Local Setup
```bash
npm install
npm run dev
```
